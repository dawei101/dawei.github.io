jquery 默认是异步返回的,如果定义函数需要ajax后的返回值，那就需要ajax 同步了  
jquery 同步返回 ：  
$.ajax(   
{   
url : &#8216;your url&#8217;,   
data:{name:value},   
cache : false,   
async : false,   
type : "POST",   
dataType : &#8216;json/xml/html&#8217;,   
success : function (result){   
do something&#8230;.   
}   
}  
);
